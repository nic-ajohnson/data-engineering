This application requires an installion of bundler and was developed using Rails 4.0.3 and Ruby 2.1.1

Steps to set up application:

1) Clone application\n
2) cd into the application directory\n
3) Run this command: bundle install\n
4) Run this command: rake db:create && rake db:migrate\n
5) To initiate server enter this command: rails server\n
6) Using a browser, navigate to localhost:3000 to view application\n
